# 🌱 KrushiRent AI
### Krushi Rent AI : Ml Based Rental And Recomendation System For Agricultural Equipment

## 📌 Overview

KrushiRent AI is an intelligent agricultural equipment rental platform designed to bridge the gap between farmers and equipment providers. The system leverages Machine Learning to recommend the most suitable farming machinery based on crop type, land size, farming activity, seasonal requirements, and equipment availability.

The platform aims to reduce farming costs, improve machinery utilization, and support precision agriculture through AI-driven recommendations.

## 💡 Problem Statement

Many farmers cannot afford expensive agricultural machinery for occasional use. Finding the right equipment at the right time is also challenging.

KrushiRent AI solves this problem by providing a digital platform where farmers can rent equipment and receive intelligent recommendations based on their farming needs.

## 🤖 Machine Learning Module

The recommendation engine analyzes multiple parameters such as:

- Crop Type
- Land Area
- Equipment Category
- Seasonal Demand
- Previous Rental Trends

Based on these factors, the system recommends the most suitable agricultural equipment.
